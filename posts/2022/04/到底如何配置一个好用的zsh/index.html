关于本文

这篇文章想要探究如何在你的电脑上配置一个好用的命令行工具 zsh, 我不会使用 ohmyzsh, 而且我想尽可能覆盖到所有的操作系统. 本教程适合 Windows, Linux, Mac 用户.

需要用到

所需 介绍
shell 你需要一点点 shell 脚本的知识
zsh Z-Shell, 一个流行的 UNIX 系统的命令行解释器
terminal 选择一个好用的终端模拟器
git 找个地方把你的配置文件保存起来, 不要弄丢了, 这里我用 git 管理配置文件.
editor 用你顺手的编辑器, 只要这个编辑器有记事本的功能就足够了

操作步骤

  1. 在你的操作系统上安装一个好用的终端模拟器, 然后安装 zsh 这个命令解释器.
  2. 配置并使用 zsh
  3. 将你的配置上传到 Github 并用 git 管理.

第一步

安装一个好用的终端模拟器

我们的要求有两点, 一是它能显示中文和英文; 二是它能显示丰富的颜色.

Windows 下我们以 Windows Terminal 为例
